## Installation 前置下載(怎麼安裝
**以下環境皆為Linux**
- 在SD卡灌入樹梅派系統
- 設定 WiFi 以及 SSH
  - WiFi:`sudo raspi-config` -> `System Option` -> `Wireless LAN`
  - SSH :`Interface Option` -> `SSH` -> 選擇 Yes

 ### 安裝軟體
- 安裝 Apache
  - `sudo apt install apache2`
